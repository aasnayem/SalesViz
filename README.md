# Retail Sales & Customer Insights Dashboard

## 📌 Project Overview
This project analyses retail sales data to uncover insights into customer behaviour, product performance, and overall business trends. The goal is to transform raw transactional data into actionable insights using data analytics and visualisation tools.

---

## 🎯 Objectives
- Identify top-performing products and categories  
- Analyse customer purchasing behaviour  
- Evaluate profitability across different segments  
- Provide data-driven recommendations for business decisions  

---

## 🛠️ Tools & Technologies
- Python (Pandas) – Data cleaning and transformation  
- Tableau – Data visualisation and dashboard creation  

---

## 📂 Dataset
- Retail sales dataset (Superstore-style data)
- Includes orders, customers, products, sales, and profit information  

---

## 🔍 Key Insights
- Office Supplies generates the highest overall revenue  
- Profitability varies significantly across categories  
- A small number of products contribute to a large portion of total sales  
- Customer spending is unevenly distributed, with high-value customers driving revenue  

---

## ⚙️ Project Workflow
1. Data Cleaning & Preparation (Python)  
2. Data Analysis (Pandas)  
3. Feature Engineering (customer segmentation, time features)  
4. Data Visualisation (Tableau Dashboard)
